📖 Overview

VLUXGEN v2.0 is a powerful all-in-one CLI toolkit for:

Ethical Hackers 🧑‍💻
Bug Bounty Hunters 🐞
Cybersecurity Students 🎓

It provides automation + recon + safe testing tools in a single interface.

🚀 Features
🔐 Wordlist Generator
🌐 Advanced Web Crawler
🔍 Parameter Finder
🧠 Google Dork Generator
💉 XSS Payload Generator (Safe)
🧬 SQL Injection Payload Generator (Safe)
🌎 Subdomain Finder
🛡️ WAF Detector
⚡ Port Scanner (Nmap-style)
🔒 Security Header Auditor
🔥 XSS Reflection Analyzer
🔁 Open Redirect Scanner






⚙️ Installation
1️⃣ Clone Repository
git clone https://github.com/cybersriram7/Vulxgen.git
cd vulxgen
2️⃣ Install Requirements
pip install -r requirements.txt
▶️ Run Tool
python vluxgen.py
